ID's are used in place of the old 'anchors' in HTML. Those anchors are the source of the 'a' in 'a href'. In order to link back to the Document Fragment, you reference it in an 'a href' tag and include the id preceded by a hash.

The Document Fragment can be found at above.

If a section of block level content (be it a paragraph, multiple paragraphs, a list, etc.) is quoted from somewhere else, you should wrap it inside a blockquote element to signify this, and include a URL pointing to the source of the quote inside a cite attribute.

Also, abbr. can be hovered over to 'see' the full word.
